# A.Ges.Co.S. (Air Gesture Control System)

An advanced real-time computer vision pipeline engineered to translate dynamic hand gestures into system-level adjustments (volume and brightness control) using low-latency execution maps.

## 🚀 Key Features & Engineering
* **Real-Time Pipeline:** Leverages Python, OpenCV, and MediaPipe for accurate hand landmark tracking.
* **Feature Engineering:** Implements mathematical vector calculations to translate 3D coordinates into precise system metrics.
* **Signal Optimization:** Designed coordinate mapping logic to filter out environmental background noise and tracking jitter.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** OpenCV, MediaPipe, PyCaw, Screen Brightness Control

## 💻 Setup and Installation

1. Clone the repository:

   git clone [https://github.com/ShreshthGupta-6/A.Ges.Co.S.git](https://github.com/ShreshthGupta-6/A.Ges.Co.S.git)
   cd A.Ges.Co.S
   
3. Install the required dependencies:

   pip install -r requirements.txt

3. Run the application:

   python main.py
